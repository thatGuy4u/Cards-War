# 🃏 Cards War Game

A simple **Cards War** game built using **HTML, CSS, and JavaScript**. This project simulates the classic card game where two random cards are drawn and compared, with the higher card winning the round.

## 🚀 Features

* 🃏 Draw two random playing cards
* 🏆 Automatically determine the winner of each round
* 🎨 Clean and responsive user interface
* ⚡ Fetches card data from Scrimba API
* 🔄 Play multiple rounds with a single click

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* Deck of Cards API

## 📂 Getting Started

Clone the repository:

```bash
git clone https://github.com/thatGuy4u/Cards-War.git
```

Navigate to the project folder:

```bash
cd cards-war
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm start
```

Open the local URL shown in your terminal (usually `http://localhost:5173`).
